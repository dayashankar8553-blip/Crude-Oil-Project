# 🛢️ Crude Oil Market Analysis Dashboard

## 📌 Project Overview
This project presents a comprehensive analysis of the global crude oil market using Power BI. It focuses on understanding price trends, supply-demand dynamics, geopolitical risks, and macroeconomic factors impacting oil prices.

The dashboard is designed to provide actionable insights through interactive visualizations and storytelling techniques.

---

## 🎯 Objectives
- Analyze crude oil price trends over time.
- Understand global supply vs demand imbalance.
- Evaluate the impact of geopolitical events on oil markets.
- Study macroeconomic indicators like inflation, interest rates, and volatility.
- Identify key oil-producing countries and their influence.
